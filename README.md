Dotfiles are managed with [GNU Stow](https://www.gnu.org/software/stow/).

```
git clone -b 1080p https://github.com/mfin/dotfiles ~/.dotfiles
cd ~/.dotfiles
stow <folder name>
```

* rxvt-unicode-256xresources, dmenu2, lemonbar-xft-git, stlarch_font
